QAPI is an API that returns a random question.

QAPI returns a random question in JSON format for you to use in your applications. There is a Python dictionary of {{total_questions}} questions and one is chosen at random and sent to the user. The user can request a specific topic or type of question to use.

Features

  • {{total_questions}} questions with more being added regularly.
  • Different types of questions
  • Different topics for questions
  • Answers
